# Test Scope – Shopify Demo

## Objective

To validate the end-to-end e-commerce workflow ensuring correct functionality, usability, and data integrity.

## In Scope

* Product browsing
* Cart operations
* Checkout process
* Payment validation
* Order confirmation

## Out of Scope

* Backend API validation
* Performance testing
* Security testing

## Risks

* Payment gateway simulation only
* Limited real-world shipping validation
