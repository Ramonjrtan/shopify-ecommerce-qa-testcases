# 🛒 Shopify Demo E-commerce QA Test Suite

## 📌 Project Overview

This repository contains manual test cases for a Shopify demo e-commerce application.

The objective is to validate end-to-end user flows including:

* Product browsing
* Cart management
* Checkout process
* Payment validation

---

## 🎯 Scope of Testing

### Functional Areas Covered:

* Storefront Navigation
* Product Catalog & Details
* Search & User Account
* Cart Functionality
* Checkout (Contact & Delivery)
* Shipping & Payment
* End-to-End Order Flow

---

## 🧪 Test Coverage

* Total Test Cases: **47**
* Includes:

  * Positive scenarios
  * Negative scenarios
  * Edge cases
  * Payment gateway validation (Bogus Gateway)

---

## 💳 Payment Testing

This project uses Shopify's **Bogus Gateway** for testing:

| Input | Result             |
| ----- | ------------------ |
| 1     | Successful payment |
| 2     | Failed payment     |
| 3     | Exception          |
